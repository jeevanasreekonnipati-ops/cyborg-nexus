# Task 1 - Cyborg Nexus

This repository contains a small web project with a static front-end and a Node.js server.

## Files

- `index.html` — main web page
- `styles.css` — project styles
- `script.js` — client-side JavaScript
- `server.js` — Node.js server
- `package.json` — project metadata and dependencies

## Dependencies

- Express
- Socket.IO

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
3. Open the app in your browser at `http://localhost:3000`

## Notes

- This project is configured as `commonjs`.
- Use `npm install` before running the server.
